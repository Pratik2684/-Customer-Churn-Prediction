# 📉 Customer Churn Prediction

Built a machine learning model to predict customer churn for a 
telecom company using Logistic Regression and Random Forest classifiers.

## 📌 Key Insights
- Customers on month-to-month contracts are 3x more likely to churn
- Contract type and tenure are top churn predictors
- Random Forest achieved 84% accuracy
- Visualized churn patterns using heatmaps and distribution plots

## 🛠️ Tools Used
Python | Pandas | Scikit-learn | Matplotlib | Seaborn | Google Colab

## 📁 Dataset
Telco Customer Churn Dataset from Kaggle

## 🚀 How to Run
1. Open the `.ipynb` file in Google Colab
2. Upload `WA_Fn-UseC_-Telco-Customer-Churn.csv` dataset
3. Run all cells
